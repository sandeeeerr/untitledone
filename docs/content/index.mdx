# UntitledOne Documentation

Welcome to **UntitledOne** - a full-stack music collaboration platform built with Next.js, Supabase, and Shadcn/UI.

UntitledOne is an open-source platform designed for remote collaboration between audio creatives. It combines file-sharing, time-based feedback, and lightweight project management â€” optimized for usability, modularity, and speed.

> Tip: See the new [Personas & How Might We](/personas) page for context.

## Key Features

- **Project Management**: Create and manage music projects with collaborators
- **File Sharing**: Upload and share audio files with version control
- **Time-based Feedback**: Add comments with precise timestamps
- **Real-time Collaboration**: Work together in real-time with other audio creatives
- **Modern UI**: Built with Shadcn/UI for a beautiful, accessible interface

## Tech Stack
## UI Preview

![UntitledOne UI](https://raw.githubusercontent.com/sandeeeerr/untitledone/feat/landing-and-docs/public/images/screenshot.png)


- **Frontend**: Next.js 15 (App Router), React, TypeScript
- **Backend**: Supabase (Auth, Database, Storage)
- **UI**: Shadcn/UI + Radix UI + Tailwind CSS
- **State Management**: TanStack Query
- **Forms**: React Hook Form + Zod
- **Internationalization**: next-intl
