# Project Structure (App)

The application uses **Next.js App Router**, **TypeScript**, and **Supabase**.

```
src/
  app/               # routes
  components/        # UI components (shadcn/ui)
  lib/               # helpers
  i18n/              # messages
supabase/            # SQL, migrations, policies
```

Recommended scripts in the app repo:
- `pnpm lint`
- `pnpm typecheck`
- `pnpm build`
